# Milestone 3 – Model Training & Clustering Pipeline

This repository contains the complete workflow for Milestone 3, including data preprocessing, clustering, pseudo-labeling, and supervised model training.

## Project Overview
The project focuses on transforming an unlabeled dataset into a risk classification pipeline using:
- Data Cleaning
- Unsupervised Clustering
- Pseudo-Labeling
- Classification Models
- Evaluation Metrics

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab

## Main Steps
1. Load & preprocess dataset  
2. Apply PCA for visualization  
3. Perform K-Means & GMM clustering  
4. Evaluate clusters (Silhouette, Davies-Bouldin, Calinski-Harabasz)  
5. Map clusters → risk levels  
6. Train multiple classification models  
7. Compare performance  
8. Select best model  
9. Perform hyperparameter tuning  

## Outputs Generated
- Cleaned dataset
- Cluster-labeled dataset
- Pseudo-labeled dataset
- Model performance tables
- Evaluation plots
